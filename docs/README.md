1.2 知识库划分

![image12.png](/media/image12.png)

运幄知识库的分为三种知识库，行业知识库,  个人知识库, 企业知识库。

行业知识库提供行业内的相关行业知识文件；个人知识库由用户个人整理上传自己的知识文件，供自己使用；

企业知识库由企业管理员整理上传，供企业内员工使用；

其所有知识库均支持检索，聊天（提问），同时支持右键选择对单/多文件，交叉知识库检索。

1.2.1 行业知识库

行业知识库由我方进行建设和管理，我方会持续更新知识库内的文件和知识内容，并将不断拓展各行业知识库。

![image23.png](/media/image23.png)

由行业管理员维护, 我们将给您划分属于您行业的相关知识库, 您可以在此处浏览您行业内的所有知识库文件

1.2.2 企业知识库

由您所属的企业管理员,  与企业管理员指定的用户一同维护。拥有权限的用户可以在此处上传专属于您企业的知识文件, 如发票，人员数据表，企业报表等。待文件解析后，可以通过本系统进行提问与检索。

![image25.png](/media/image25.png)

企业管理员可以在设置——企业管理中添加员工子账户，并进行子账户权限管理。

1.2.3 个人知识库

由您自己管理，您可以在此处上传属于你自己的文件。
